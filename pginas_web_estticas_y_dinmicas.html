¿Conoces lo que significa "página WEB"?

Se entiende como página WEB a un documento o información electrónica que se encuentra adaptada "WWW" cuyas siglas refieren a "World Wide Web". 

Incluye variedad de contenido y tecnología a las cuales se puede acceder mediante un navegador web.
